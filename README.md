# Stock Prediction Portal

This is a full-stack web application built using Django REST Framework for the backend and React.js for the frontend.

At the current stage, the project implements user authentication using JWT (JSON Web Tokens) with Simple JWT. In the future, this application will be extended into a complete stock prediction dashboard.

## Current Features

• User registration using DRF APIs
• User login using JWT access and refresh tokens
• User logout (token-based)
• Authentication handled using Simple JWT
• React frontend integrated with backend APIs

## Future Scope

• Stock market data integration
• Stock price prediction using machine learning
• Interactive dashboard with charts
• Historical and predicted stock analysis
• User-specific dashboards

This project demonstrates:
